# Tạo bảng kê hóa đơn, chứng từ, hàng hóa, dịch vụ mua vào tự động

<p>taihoadon.online hỗ trợ tạo bảng kê hóa đơn, chứng từ, hàng hóa, dịch vụ mua vào <strong>(Mẫu 01-1/HT theo thông tư 80)</strong> tự động từ dữ liệu từ dữ liệu hóa đơn điện tử</p>

<img src="/blog/bk-01-1.png" alt="Developer mode" />

## B1. Tải file excel từ trang taihoadon.online

1. Truy cập trang taihoadon.online, nhập thông tin tài khoản, mật khẩu, ngày lập hóa đơn
2. Chọn hóa đơn mua vào
3. Nhấn **Tải dữ liệu** để tải file excel

---

## B2. Kiểm tra thông tin trong sheet "Bảng kê mua vào"
- Trong file excel tải về, mở sheet <strong>BK_mua_vào_TT80</strong>
- Trong file đã bao gồm đầy đủ thông tin theo mẫu <strong>01-1/HT thông tư 80</strong>, bảng kê hóa đơn, chứng từ, dịch vụ mua vào. Vui lòng kiểm tra lại thông tin để chắc chắn đúng.
